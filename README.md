# ✨ ForgeBase
- **Descrição:** `Uma simples base para iniciantes no forgescript.`
# 🍂 Instalação
**1. Clone o Repositório:**
```bash
git clone https://github.com/eternalsouza/ForgeBase.git
cd ForgeBase
```
**2. Instale as Dependências:**
```bash
npm install
```
**3. Configure o .env:**
```env
CLIENT_TOKEN=seu_token
```
# 🪐 Como Iniciar
**1. Inicie Normalmente:**
```bash
npm run start
```
# 🌻 Recomendações
- **Node: v25.8.0**
- **Utilizar: [Discloud](https://discloud.com/)**
